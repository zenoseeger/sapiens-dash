Configuração inicial

Siga os 3 passos abaixo para conectar o dashboard ao Google Sheets.
Leva menos de 10 minutos e só precisa fazer uma vez.

1

Criar a planilha no Google Sheets

Acesse sheets.google.com e crie uma planilha nova.
Nomeie ela como Sapiens Leads.
Copie o ID da URL — é o trecho longo entre /d/ e /edit:
docs.google.com/spreadsheets/d/SEU_ID_AQUI/edit

2

Criar o Web App (Google Apps Script)

Na planilha, clique em Extensões → Apps Script.
Apague o código existente e cole o código abaixo, depois clique em Implantar → Nova implantação → Web App.
Em "Quem pode acessar" escolha Qualquer pessoa. Clique em Implantar e copie a URL gerada.

3

Conectar e importar dados históricos

Cole os dois valores acima e clique em Conectar. O dashboard vai gravar os dados históricos automaticamente na planilha.
